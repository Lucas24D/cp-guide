Introduction / Introdução

The objective of this guide is to help everyone who is practicing for competitive programming contests. Although there are many guides that contains lots of information about competitive programming, it still hard to find somewhere a compilation of tutorials from different places. To make it easier to find and study a specific topic, here we present a list with links to lessons and explanations for many of them. We also list extra resources that can help while practicing. You can also contribute to this guide, adding a link to your tutorials or to other non-listed materials here by pull request.

O objetivo deste guia é ajudar aqueles que estejam praticando para competições de programação. Apesar de existirem diversos guias que contém muitas informações acerca de programação competitiva, ainda é difícil encontrar um compilado de tutoriais de diferentes lugares. Para facilitar a busca e o estudo de um tópico específico, apresentamos aqui uma lista contendo links para aulas e explicações para muitos deles. Também listamos recursos extras que podem ajudar durante o treinamento. Você também pode contribuir com este guia, adicionando um link para o seu ou outro material não listado aqui via pull request.

Extra Resources / Recursos Extras

Here we list some existing resources and tools that complements our guide and could be helpful to anyone who is studying competitive programming.

Aqui nós listamos algumas ferramentas e recursos que complementam nosso guia e podem ajudar todos que estejam estudando para programação competitiva.

  • An awesome list for competitive programming!

    • This post published on Codeforces from user Inishan (Jasmine Chen) contains lots of informations, such as places to practice, books about competitive programming and math, tutorial websites, repositories with implementations etc. It's one of the best resources available that can help newcomers and advanced competitors to find what they need.

    Este post publicado no Codeforces pelo usuário Inishan (Jasmine Chen) contém diversas informações, como locais para praticar, livros sobre programação competitiva e matemática, websites contendo tutoriais, repositórios com implementações etc. É um dos melhores recursos disponíveis que pode ajudar os novatos e os competidores mais avançados a encontrar aquilo que precisam.

  • URI Online Judge

    • A website made in Brazil, however it is also available in English, that has a large variety of problems to solve. It is a good platform for newcomers, due to their beginner's category containing lots of easy problems for those who are new in programming.

    Um website feito no Brasil, porém também disponível em Inglês, que possui uma grande variedade de problemas para resolver. É uma boa plataforma para novatos, por conta de sua categoria iniciante contendo diversos problemas fáceis para aqueles que programam há pouco tempo.

  • CSES-Code Submission Evaluation System

    • This website contains some easy and classic problems with short statements, divided by categories. Although it has less problems than most of the websites listed on the first link, it is good for practice new learned techniques and get to know some important topics.

    Este website contém problemas fáceis e classicos com enunciados curtos e diretos, divididos por categorias. Apesar de possuir menos problemas que muitos dos sites listados no primeiro link, é um bom lugar para praticar novas técnicas aprendidas e conhecer alguns tópicos importantes.

  • Neps Academy

    • Brazilian Platform that also contains easy and classic problems divided by categories. It also has lessons to basic concepts and topics. It's a good alternative to CSES for Brazilian programmers.

    Plataforma Brasileira que também contém problemas fáceis e clássicos divididos por categorias. Também possui aulas sobre conceitos e assuntos básicos e iniciais. É uma boa alternativa ao CSES para programadores brasileiros.

  • A2 Online Judge and Problem Classifier

    • Good problem classifiers to help searching for problems from a specific category. The first one categorizes problems from various sites and is also an problem solving platform. The second one is only a tool to find specific SPOJ problems and it is nice to use it while navigating through this online judge, which can be a little confusing at first glance without any help.

    Bons classificadores de problema para ajudar a encontrar problemas de uma categoria específica. O primeiro categoriza problemas de diversos sites e também é uma plataforma de resolução de problemas. O segundo é apenas uma ferramente utilizada para encontrar problemas específicos no SPOJ e é bom usá-la enquanto navega por esta plataforma, que pode ser confusa à primeira vista sem nenhuma ajuda.

  • Competitive Programming Problems' Solution Library

    • A repository containing solutions for more than 1000 problems until now (mostly from Codeforces) from my own training sessions. It is regularly updated with new problems as soon as I solve them. It is also open for contribution, especially from people who use languages other than C++ (used to solve almost all problems found there), to make it bigger and offer solutions for problems from more platforms, solved using more than one programming language.

    Um repositório contendo soluções para mais de 1000 problems até o momento (a maioria do Codeforces) de minhas próprias sessões de treino. É regularmente atualizado com novos problemas assim que os resolvo. É também aberto para contribuição, especialmente de pessoas que usam outras linguagens além do C++ (usada para resolver a maior parte dos problemas encontrados lá), para fazê-lo maior e oferecer soluções para problemas de mais plataformas, resolvidos usando mais de uma linguagem de programação.

List of Topics / Lista de Tópicos

Here we present the list of topics. Each one is divided into subtopics with tutorials found around the web. All the materials listed here are in English or in Portuguese. You don't need to follow it in order. Also, some subtopics from one category can be prerequisite to subtopics from other categories. We assume that you are familiar with basics concepts of programming (like Loops, Arrays, Recursion, Complexity Calculation, Binary Search, etc) and know how to use some structures, like Set, Map, Stack and Queue from C++. Problems for the classic and basic concepts can be easily found on CSES and Neps Academy (listed above). For the more advanced topics, most of the tutorials already have suggested problems to solve. Also, more problems can be found using problem classifiers listed above on other platforms.

Aqui apresentamos a lista de tópicos. Cada um deles é dividido em subtópicos com tutoriais encontrados pela internet. Todos os materiais listados aqui estão em Inglês ou em Português. Não é necessário seguir a lista em ordem. Além disso, alguns subtópicos de uma categoria podem ser pré-requisitos para subtópicos de outras categorias. Assumimos que o usuário deste guia está familiarizado com conceitos básicos de programação (como Laços de Repetição, Arrays, Recursão, Cálculo de Complexidade, Busca Binária, etc) e sabe como utilizar algumas estruturas, como Set, Map, Stack e Queue do C++. Problemas para os conceitos clássicos e básicos podem ser facilmente encontrados no CSES e no Neps Academy (listados acima). Para os tópicos mais avançados, a maior parte dos tutoriais já contém problemas sugeridos para resolver. Além disso, mais problemas podem ser encontrados usando os classificadores de problemas citados acima em outras plataformas.

  • Dynamic Programming / Programação Dinâmica
  • Graphs / Grafos
  • Math, Number Theory and Game Theory / Matemática, Teoria dos Números e Teoria dos Jogos
  • String Processing / Processamento de Strings
  • Geometry / Geometria
