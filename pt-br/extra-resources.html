Recursos Extras

Aqui são listados algumas ferramentas e recursos que complementam nosso guia e podem ajudar todos que estejam estudando para programação competitiva.

  • An awesome list for competitive programming!

    • Este post publicado no Codeforces pelo usuário Inishan (Jasmine Chen) contém diversas informações, como locais para praticar, livros sobre programação competitiva e matemática, websites contendo tutoriais, repositórios com implementações etc. É um dos melhores recursos disponíveis que pode ajudar os novatos e os competidores mais avançados a encontrar aquilo que precisam.

  • All of the good tutorials found on codeforces

    • Este post do usuário Underdog_eagle contém uma lista com um grande número de tutoriais encontrados na plataforma Codeforces. Esses tutoriais contém informações extras sobre alguns tópicos, algoritmos e truques que podem ajudar todos os competidores.

  • URI Online Judge

    • Um website feito no Brasil, porém também disponível em Inglês, que possui uma grande variedade de problemas para resolver. É uma boa plataforma para novatos, por conta de sua categoria iniciante contendo diversos problemas fáceis para aqueles que programam há pouco tempo.

  • CSES-Code Submission Evaluation System

    • Este website contém problemas fáceis e classicos com enunciados curtos e diretos, divididos por categorias. Apesar de possuir menos problemas que muitos dos sites listados no primeiro link, é um bom lugar para praticar novas técnicas aprendidas e conhecer alguns tópicos importantes.

  • Neps Academy

    • Plataforma Brasileira que também contém problemas fáceis e clássicos divididos por categorias. Também possui aulas sobre conceitos e assuntos básicos e iniciais. É uma boa alternativa ao CSES para programadores brasileiros.

  • A2 Online Judge e Problem Classifier

    • Bons classificadores de problema para ajudar a encontrar problemas de uma categoria específica. O primeiro categoriza problemas de diversos sites e também é uma plataforma de resolução de problemas. O segundo é apenas uma ferramente utilizada para encontrar problemas específicos no SPOJ e é bom usá-la enquanto navega por esta plataforma, que pode ser confusa à primeira vista sem nenhuma ajuda.

  • Competitive Programming Problems' Solution Library

    • Um repositório contendo soluções para mais de 1000 problems até o momento (a maioria do Codeforces) de minhas próprias sessões de treino. É regularmente atualizado com novos problemas assim que os resolvo. É também aberto para contribuição, especialmente de pessoas que usam outras linguagens além do C++ (usada para resolver a maior parte dos problemas encontrados lá), para fazê-lo maior e oferecer soluções para problemas de mais plataformas, resolvidos usando mais de uma linguagem de programação.
